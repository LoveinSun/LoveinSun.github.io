1390. 四因数

给你一个整数数组 nums，请你返回该数组中恰有四个因数的这些整数的各因数之和。如果数组中不存在满足题意的整数，则返回 0 。

我解

复杂度分析

  • 时间复杂度：$$O(N \sqrt[2]{C}）$$，其中 N 是数组 nums 的长度，C 是数组 nums 中元素值的范围，在本题中 C 不超过 10^510
    5。

  • 空间复杂度：O(1)O(1)。

错误：

没有考虑两个因数相等的情况，也就是18，19行

他解

枚举

复杂度分析

  • 时间复杂度：$$O(N \sqrt[2]{C}）$$，其中 N 是数组 nums 的长度，C 是数组 nums 中元素值的范围，在本题中 C 不超过 10^510
    5。

  • 空间复杂度：O(1)O(1)。

官解

使用筛法

思路就是使用筛法过滤出所有的需要的素数，然后使用它们构建四因数

然后涉及到的就是四因数有何种性质使得其可以用质数进行构建。
